BYU Learning Suite Responsive Design

I worked with BYU's development team at The Center for Teaching and Learning to create mobile and responsive designs for their school-wide learning management system, Learning Suite.

Creating responsive layouts out of existing elements can be tricky. We made innovating changes using some clever CSS.

I created mockups and presentations explaining the proposed UI changes. The design is expected to hit production this fall.
